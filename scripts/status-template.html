📋 配置步骤：
  1. 复制上面的公网隧道地址
  2. 打开 Cloudflare Dashboard → Pages → 你的项目
  3. Settings → Environment Variables
  4. 更新 VITE_API_BASE 的值为这个地址
  5. 回到 Deployments，点最近一次部署右边的 "Retry deployment"
  6. 等待重新构建完成（约 1-2 分钟）
  7. 让朋友访问你的 Pages 域名（如 small-game.pages.dev）
